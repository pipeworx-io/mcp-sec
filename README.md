Part of the [Pipeworx](https://pipeworx.io) open MCP gateway.

## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "sec": {
      "url": "https://gateway.pipeworx.io/sec/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use sec
```
